# Scalable-REST-API-with-Authentication-Role-Based-Access
# Scalable Backend & Frontend Project

This project is a **full-stack web application** demonstrating a scalable backend with authentication, role-based access control, CRUD APIs, and a basic frontend UI to interact with the APIs.

---

## Project Overview

- **Backend:** Node.js + Express
  - User registration & login with password hashing and JWT authentication
  - Role-based access control (user vs admin)
  - CRUD APIs for a secondary entity (e.g., tasks, notes, products)
  - Input validation and error handling
  - API versioning (`/api/v1/...`)
  - Database schema using MongoDB
  - API documentation via Swagger/Postman

- **Frontend:** React.js
  - User registration and login forms
  - Protected dashboard accessible only with JWT
  - CRUD interface for the secondary entity
  - Display error and success messages from API responses
